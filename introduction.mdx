DiegoMusic is a free, open-source music streaming app for iOS and Android. It works by connecting a React Native mobile client to a self-hosted Node.js backend, which extracts audio and video from YouTube using yt-dlp. You get a Spotify-like experience — search, favorites, offline playback, and lock-screen controls — with no subscription, no account, and no third-party streaming service required.

<Note>
  DiegoMusic is self-hosted software. The mobile app cannot function on its own — you must deploy the backend server before installing the app on your device. See the [deployment guide](/deployment/backend) to get started.
</Note>

## Architecture overview

DiegoMusic is split into two independent parts that work together. The mobile app handles the user interface and media playback. The backend handles all communication with YouTube, including search, audio extraction, and video streaming.

<CardGroup cols={2}>
  <Card title="Mobile app" icon="mobile" href="/quickstart">

  <Card title="Backend server" icon="server" href="/deployment/backend">

## Technology stack

| Component | Technology | Version |
|---|---|---|
| Mobile app | Expo / React Native | 3.0.0 |
| Backend server | Node.js + Express | 1.1.0 |
| Audio/video extraction | yt-dlp | latest |
| Audio processing | FFmpeg (via ffmpeg-static) | bundled |
| Containerization | Docker + Docker Compose | — |
| Routing | Expo Router | ~6.0.23 |
| Language | TypeScript (app), JavaScript ESM (backend) | — |

The backend Dockerfile installs FFmpeg and yt-dlp at build time, so the Docker image is fully self-contained. No external media services or API keys are required beyond optional YouTube browser cookies.
